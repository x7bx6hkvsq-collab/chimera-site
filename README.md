# chimera-site
オリジナルのキメラを、一枚絵と短い設定で投稿するための静かな投稿サイト。
このサイトは、オリジナルのキメラを  
一枚絵＋任意の説明文で投稿するための場所です。

- 絵の上手さは問いません  
- 名前は無名でも構いません  
- 説明欄には、モチーフにしたものがあれば書いても、書かなくても構いません  

---

### サイトの空気感

- 静か  
- 通り過ぎる人が多い  
- 常連は別に偉くない  
- 作者と閲覧者の距離が遠い  
- 残るけれど、絡まない  

---

### やらないこと

- いいねなどの数値評価は置きません  
- ランキング・おすすめ・急上昇は作りません  
- フォロワー数は表示しません  
- コメントはデフォルトでオフです  
- 議論が起きる仕組みは作りません  
- バズを前提にしません  

この場所は、  
創作物が静かに並び、人が通り過ぎるためのサイトです。
